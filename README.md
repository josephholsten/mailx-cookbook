Description
===========

Installs mailx.

Requirements
============

### Platform ###

The recipe has only been tested on Ubuntu, but should work on recent
Debian variants.

Please [report][issues] any additional platforms so they can be added.

Attributes
==========

None.

Usage
=====

Simply apply the `mailx::default` recipe to your run list!
